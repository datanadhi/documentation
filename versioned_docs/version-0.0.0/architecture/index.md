# Architecture

This section documents the architecture of Data Nadhi components.

## Components

### [SDK (Python)](./sdk-python/)
The Python SDK for structured logging with rule-based pipeline routing.

### [EchoPost](./echopost/)
Local buffer agent for reliable log delivery during server outages.
